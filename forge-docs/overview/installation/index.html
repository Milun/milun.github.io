Installation

To develop with Forge, you will first need to install a few third-party tools:

  • NodeJS
  • VSCode

NodeJS setup​

Download and install NodeJS from the link above. Once it completes, check that it installed successfully by:

  1. Open a Command Prompt window (shortcut: Windows + R, then type in cmd).
  2. Run the npm -v command.
  3. If Command Prompt shows a number, you've installed NodeJS correctly! If not, you will need to add npm to your environmental Path variables first.
  4. Open a new Command Prompt window, and then try step 2 again.

VSCode setup​

Once you have downloaded and installed the above, open VSCode. You will need to install a few extensions (they will simplify development). Press Ctrl + Shift + X to open the Extensions window.

Required extensions​

Simply copy + paste the following into the searchbar and install what it gives you:

RoscoP.ActiveFileInStatusBar
Makes it easier to tell which file you've currently got open.

VisbyDev.folder-path-color
Colourizes the folders in the explorer.

PKief.material-icon-theme
Adds custom icons to the folders in the explorer.

vincaslt.highlight-matching-tag
Makes writing HTML easier to visualize.

Required JSYNT extension​

The JSYNT extension is custom made for this project, and needs to be installed a bit differently.

  1. Download the extension file.
  2. Click and drag the .vsix file you downloaded onto the Extensions area in VSCode.
  3. You should get a popup saying "extension installed" in the bottom-right.
  4. Restart VSCode. You can easily do this by pressing Ctrl + Shift + P, and typing in "Developer: Reload window".

Optional extensions​

These are extensions I have found personally useful. They're not essential for development.

ms-vscode.notepadplusplus-keybindings
Improves VSCode's hotkeys (I never really liked the defaults)

swyphcosmo.spellchecker
Adds (somewhat reliable) spellchecking.

VSCode settings​

The following settings should be set in your VSCode to make development easier. You'll only need to do this once.

Increasing font-size​

In my opinion, the default VSCode font size is too small. Here's how you change it globally:

  1. In VSCode, press Ctrl + Shift + P, and type in 'Open user settings'.
    Select the option which states: Preferences: Open user settings (not the JSON one).

  2. In the window that appears, type in 'font' in the search bar. Change the number in 'Editor: font size' to 18 (or higher).

Changing default terminal (command prompt)​

Grunt tasks run more reliably when called through a Command Prompt terminal. To ensure that that's what VSCode uses, please do the following:

  1. In VSCode, press Ctrl + Shift + P, and type in 'Terminal profile'.
    Select the option which states: Terminal: Select Default Profile.

  2. Select the 'Command Prompt' option.

This will come into play on the next page.
