Creating your first course

This guide will cover how to create a new Forge course from scratch. Please ensure you have read and followed the instructions in installation before proceeding.

1. Create a directory to store all of your courses​

Create a folder anywhere on your machine (I recommend C:/Users/USERNAME/Documents/*), and give it a good name (I recommend Forge).

2. Download and run the Forge installer​

Download the Forge installer. Place it within the folder you just created, and then run it.

The program will walk you through the installation process (which will take a few minutes).

important

Be sure to choose a good, short name for your course! I've used FRG01 as an example.

note

The first time you run this will likely take about 5 minutes while all the node modules are downloaded. This is a one-time delay, and subsequent courses will install much faster.

Once all that is done, your folder should look like this:

3. Navigate into the src/course/en/content directory.​

Forge requires a lot in order to run, but when developing courses with it, you're only interested in one directory: the course/content directory.

It located in the following path: COURSENAME/src/course/en/content/. Navigate into it.

4. Create your first block.​

Your course currently has 0 blocks in it (and will not compile).

Run the __create-block.bat program, and it will automate creating a new block for you. Create one with the ID of 1.

5. Start grunt dev and grunt server inside of VSCode.​

You can now run the grunt server and grunt dev tasks to test your course. While both are running, you can view the course on http://localhost:9001/.

If you are confused on how to do this step, see the Grunt tasks page.

6. Open the project in VSCode.​

Run the __content.code-workspace file. If you have VSCode installed (and you should) this will open it in a special "project" view.

7. Make changes to your block.​
