Color-changing SVGs

Forge's appearance is very vector-styled; featuring many lines and geometric shapes. As such, you will likely find yourself using flat colored SVGs as decorative images, or as clickable image buttons in templates like the Hotgraphic.

The above component has parts of the SVG go from Forge's default red color (which is used to indicate incomplete interactions), to its default green color (which is used to indicate a complete interaction).

To use this feature in your content, you only need to create one SVG in a very specific way.

How to make a color-changing SVG​

Step 1 - Find your SVG​

Any SVG will do, but try to avoid overly complex ones. Complex SVGs will cause your site to lag. If you absolutely must use a complex SVG, convert it to two PNGs (one incomplete state, one complete state), and then use the "scrVisited" property instead (see the individual templates for more information).

Step 2 - Open the SVG in Illustrator, and set it to RGB mode​

Do not skip this step! Go to File > Document Color Mode > RGB Color in Illustrator.

Step 3 - Decide which parts of the image you want to change colors​

This can be anything as long as it looks good. In my example, I'm going to make his beard color-changing.

Step 4 - Recolor those parts to shades of magenta​

A "shade of magenta" is any color that has a hex code following this pattern: #XXYYXX. In essence, any color which has the same red & blue value, that is also larger than its green value, is considered "magenta" by Forge.

If Forge detects these colors in an SVG, it will automatically map them to the corresponding shade of red & green that's used within Forge. Here's just some examples of what magentas map to what shades:

MagentaRedGreen
#ffddff
#ff66ff
#ff00ff
#663366
tip

#ff00ff is the hex code for "true" magenta, and maps to the default shade of Forge red and Forge green when used.

tip

When using Illustrator's color picked to choose a magenta, type in #ff00ff into the hex entry field, then click on Only Web Colors. Every major chunk of color you see presented here is a valid shade of magenta.

Step 5 - Save your SVG​

That's all that needs to be done. Forge will automatically detect the magenta colors if you followed these steps correctly.

Troubleshooting​

"My SVG is still showing up with magenta colors when I view it"​

Check that your Illustrator is in RGB mode. If it's not, switch to it, then set your colors again (in CYMK mode, it will convert your colors to different hex codes after you save them; which is why you need to switch modes).

"It's still showing magenta"​

If all else fails, try rebooting grunt dev (it may have missed a trigger on your save).

"What if I want to use magenta in my image without it changing colors?"​

Magenta is generally seen as a gaudy and unprofessional color to use in most cases. However, if you absolutely must use it, simply make sure that the colors you use have mismatching R & B colors. Even a single digit off will disable the color-changing feature.
