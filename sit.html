SIT Training Packages

E-learning for tourism, travel and hospitality industries

Web-based training packages; designed with a desktop/tablet first approach.

I was the primary developer for this project, and created a JavaScript framework that automated the content's design, interactive elements, and LMS (SCORM) integration.

To streamline eBook development (and make it accessible to the majority of my team), I designed the framework to allow content creation with only an entry level of HTML and JSYNT knowledge, with the option for custom functionality depending on coding skill.

This data was then inserted into a series of interactive page templates made with Backbone.js, Handlebars and LESSCSS, which were then built into distributable SCORM packages using a multitude of Grunt.js tasks.

Due to the fixed size of the content, the greatest challenge I faced when designing each template was fitting text and assets of various dimensions without overflowing its bounds. I solved this by creating a lightweight 3×3 grid plugin that would allow for full control over where each element was placed, without the need for custom CSS. Every layout in the examples above is created using this plugin.

To date, my framework has been used to create over 100 units.
